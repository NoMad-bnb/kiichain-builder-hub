# KiiChain Builder Hub

A comprehensive developer guide for building on KiiChain Testnet Oro. Covers smart contract deployment, RWA, Stablecoins, FX, CrediFi, and PayFi modules with beginner-friendly explanations and production-ready code examples.

---

## Live Site

[kiichain-builder-hub](https://nomad-bnb.github.io/kiichain-builder-hub/)

---

## What This Covers

- RWA Protocol (T-REX, CosmWasm, Identity Registry)
- Stablecoin Layer (TokenFactory, Fee Abstraction, Gas with Stablecoins)
- FX Module (Cross-border payments, IBC, 24/7 Liquidity)
- CrediFi Protocol (TVL Loans, Invoice Financing, Credit Pools)
- PayFi Module (Paymaster, Payment Scheduling, Merchant TX)
- Smart contract deployment via Hardhat, Remix, Foundry, and CosmWasm
- KiiJS SDK and Rust SDK usage
- All network endpoints for Testnet Oro (Sentry 1 and Sentry 2)
- Precompiled contracts reference
- Testnet faucet guide
